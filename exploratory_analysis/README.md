# Análise Exploratória (Exploratory Analysis)

Pasta dedicada à análise descritiva e exploratória inicial da base de dados do SCR (Sistema de Informação de Crédito). 

Contém os notebooks Jupyter (como o `analise_datasets.ipynb`) utilizados para:
- Compreender a distribuição das variáveis numéricas e categóricas.
- Identificar as proporções de inadimplência e o desbalanceamento natural da base.
- Explorar possíveis *outliers* e valores ausentes.
- Levantar *insights* visuais (gráficos) que serviram de base para guiar as regras de engenharia de atributos (Feature Engineering) e o saneamento dos dados na etapa de pré-processamento.
